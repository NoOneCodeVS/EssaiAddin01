Nom de l’auteur du document

Instructions

vous devez ajouter ou modifier le nom de l’auteur du document dans les propriétés/méta-données de Word. ⚠️ Le nom ne doit pas être votre prénom, mais un nom générique de groupe de travail : « Groupe Projet Alpha ».

Indice

Ajoutez un auteur
  • Cliquez sur Fichier → Informations.
  • À droite, vous verrez la zone Propriétés → cliquez sur Afficher toutes les propriétés
  • Dans Auteur, ajoutez le nom

Le nom de l’auteur n’est pas correct. Veuillez réessayer.
